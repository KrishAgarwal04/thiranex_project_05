
A production-ready full-stack web application featuring a modular architecture, client-side routing, performance optimization, and live deployment on a modern hosting platform.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies :
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
